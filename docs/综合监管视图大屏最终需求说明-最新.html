综合监管视图大屏最终需求说明（最新）

版本：V1.1（最新确认） | 日期：2026-02-28 | 端：WEB端（不含移动端）

1. 建设目标

  • 整合畜牧管理、银行金融管理、保险管理三大模块关键指标，形成统一监管大屏。
  • 实现“总览监测 + 空间分布 + 风险预警 + 处置进度”一屏展示。
  • 支持按养殖场筛选后的联动查看。

2. 页面布局（最终）

区域要求说明
顶部标题区标题居中，双侧霓虹线延伸，右上角时间保持整体绿色科技风，不切换主色调
左侧栏固定宽度 400px，3 张卡片总览与健康、金融信息
中间地图自适应剩余宽度，中国地图主画布仅保留养殖场筛选
右侧栏固定宽度 400px，3 张卡片预警、整改进度、异常列表

3. 顶部标题区需求

  • 主标题：综合监管视图。
  • 副标题：养殖主体全景、风险预警、金融动态一屏联动。
  • 标题无边框、无厚重背景，保持简洁协调。
  • 两侧霓虹线靠近标题并向页面边缘延伸，支持流光效果。
  • 右上角显示时间与刷新状态（示例：2026-02-28 09:30 | 实时刷新）。

4. 中间地图区需求（最终）

4.1 地图与筛选

  • 地图采用中国地图。
  • 地图区仅保留一个筛选项：养殖场树形筛选。
  • 地图支持缩放/拖拽浏览。

4.2 图层要求（最终）

  • 仅保留图层：养殖场分布。
  • 已去除图层选择控件。
  • 已去除地图图例及其对应图例分组。

4.3 地图展示内容

  • 养殖场点位分级展示：正常 / 预警 / 高风险。
  • 点位支持名称标注与提示信息（tooltip）。
  • 地图底部保留简短交互提示文案。

5. 左侧三卡片（固定）

卡片名称核心内容
左1养殖主体全景存栏健康率、防疫完成率、贷款履约率、在保覆盖率
左2健康防疫已接种、未接种、即将过期
左3金融动态监管贷款管理、投保管理、理赔管理核心指标

6. 右侧三卡片（固定）

卡片名称核心内容
右1风险预警管理预警列表（等级、时间、内容） + 按钮：设置风险阈值（primary）
右2整改处理进度通知发送率、3日内处置完成率、超期未处理占比
右3异常列表养殖场、异常类型、风险等级、触发时间、处置状态、通知状态

7. 权限与数据范围

  • 权限按农场组织管理：普通用户仅看本农场数据。
  • 管理员可查看全部农场数据。
  • 地图与左右卡片数据遵循同一权限口径。

8. 交互联动要求

  • 切换养殖场筛选后：地图点位、左右卡片、异常列表同步刷新。
  • 地图点位 hover 显示信息，点击可扩展详情（后续迭代）。
  • 风险预警卡片支持快速进入阈值配置（按钮入口已保留）。

9. 验收标准（最新）

  • 左右栏均为 400px，且均为 3 卡片布局。
  • 中间为中国地图，仅显示养殖场分布图层。
  • 地图不出现图层选择与图例模块。
  • 风险预警管理仅保留“设置风险阈值”按钮（primary）。
  • 标题区风格协调：无边框，霓虹线贴近标题并可见流光效果。
