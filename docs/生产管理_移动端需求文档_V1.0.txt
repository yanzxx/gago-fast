生产管理（移动端）需求文档 V1.0

一、文档信息
1. 模块名称：生产管理（移动端）
2. 子功能：项圈登记、登记记录查询
3. 适用端：snowy-mobile
4. 版本：V1.0
5. 日期：2026-03-06

二、建设目标
1. 在手机端实现项圈登记全流程电子化。
2. 支持登记记录多维筛选查询、完整档案查看与 Excel 导出。
3. 保证项圈编号唯一，减少重复登记和错误数据。

三、功能范围
1. 项圈登记
2. 批量登记
3. 登记记录查询
4. 登记详情（完整档案）
5. 导出任务

四、页面规划（移动端）
1. 生产管理首页（入口页）
2. 项圈登记页（单条登记）
3. 项圈批量登记页（模板导入/批量提交）
4. 登记记录列表页（多维筛选）
5. 登记详情页（完整档案）
6. 导出任务页（导出进度与下载）

五、详细功能需求
5.1 项圈登记（单条）
1. 支持 3 种编号录入模式：
   - 自动生成
   - 规则生成（前缀+日期+流水）
   - 手工录入
2. 必传图片：
   - 骆驼照片
   - 项圈照片
3. 支持拍照/相册上传、图片预览、替换、删除。
4. 提交前执行编号唯一性校验；冲突时禁止提交并提示冲突编号。
5. 字段建议：项圈编号、登记日期、组织/养殖场、骆驼标识、骆驼照片、项圈照片、备注。

5.2 批量登记
1. 支持下载导入模板。
2. 支持上传批量文件并进行校验。
3. 返回失败明细（行号、失败原因、冲突编号）。
4. 支持仅重试失败项。

5.3 登记记录查询
1. 多维筛选条件：
   - 项圈编号
   - 登记时间范围
   - 组织/养殖场
   - 操作人
   - 状态
2. 支持关键词模糊查询。
3. 列表展示核心字段和缩略图。
4. 支持分页加载、下拉刷新。

5.4 登记详情（完整档案）
1. 展示完整登记信息。
2. 展示图片与操作信息。
3. 支持图片大图预览。

5.5 Excel 导出
1. 按当前筛选条件发起导出。
2. 采用异步导出任务。
3. 导出完成后可下载文件（移动端下载链接或系统通知）。
4. 记录导出日志（导出人、时间、筛选条件）。

六、业务规则
1. 编号不能为空。
2. 编号必须唯一。
3. 骆驼照片、项圈照片为必填。
4. 批量提交按条返回结果，不因单条失败阻断全部。
5. 删除/作废记录需保留审计痕迹。

七、移动端交互要求
1. 单手操作优先，主操作按钮固定在底部。
2. 拍照上传流程不超过 2 步。
3. 校验错误需就近提示（字段下方+顶部提示）。
4. 弱网场景支持失败重试，建议支持草稿暂存。

八、非功能需求
1. 查询响应：常规筛选下 <= 2 秒。
2. 安全：按角色控制新增、查询、导出权限。
3. 稳定：导出任务失败需可重试并可追踪。
4. 审计：登记、编辑、导出均记录日志。

九、验收标准
1. 移动端可完成单条登记并上传两类必传图片。
2. 三种编号模式均可登记，重复编号会被拦截。
3. 列表支持多条件组合筛选且结果准确。
4. 详情页可查看完整档案。
5. 可在手机端发起并获取 Excel 导出结果。

十、待确认项
1. 三种编号模式最终规则细节。
2. 批量模板字段与大小限制。
3. 完整档案字段范围。
4. 导出字段清单与脱敏规则。
