贷款管理需求说明（简版）

版本：V1.0（最小可用版本）

范围：仅WEB端；不包含移动端；本期不做审核操作

1. 建设目标

  • 实现贷款申请资料在线查询
  • 展示贷款审批节点状态与进度
  • 按农场维度控制数据可见范围

2. 功能范围（第一期）

  • 贷款申请分页查询
  • 贷款申请详情查看
  • 流程节点状态查看（时间线）

3. 页面结构

3.1 整体布局

  • 上方：筛选区
  • 下方：申请列表
  • 详情：弹窗或抽屉（含申请信息与节点进度）

3.2 筛选项

字段类型说明
养殖场下拉管理员可选，普通用户按权限自动限制
申请单号输入框支持精确或模糊匹配
申请人输入框模糊查询
贷款产品下拉/输入按产品名称筛选
申请状态下拉见状态定义
申请时间日期范围起止时间筛选

3.3 列表字段

字段说明
申请单号业务唯一编号
养殖场所属农场
申请人申请主体
贷款产品关联金融产品
申请金额金额字段
申请期限月数
当前节点如受理/初审/复核/放款
申请状态当前状态
申请时间提交时间
操作详情

4. 详情展示内容

4.1 申请基本信息

  • 申请单号、农场、申请人、贷款产品
  • 申请金额、申请期限、贷款用途

4.2 申请资料信息

  • 身份证明材料
  • 养殖证明材料
  • 抵押/担保相关附件

4.3 节点进度信息

  • 受理、初审、复核、放款等节点状态
  • 每个节点的处理时间与备注

5. 状态定义（建议）

状态码状态名称说明
SUBMITTED已提交申请已创建，待处理
UNDER_REVIEW审核中处于流程节点处理中
REJECTED已拒绝审批不通过
APPROVED已通过审批通过待放款
DISBURSED已放款放款完成

6. 权限与数据范围

  1. 按农场组织维度授权。
  2. 普通用户仅查看本农场申请数据。
  3. 管理员可查看全部农场数据。
  4. 本期仅开放“查看”权限，不开放审核操作按钮。

7. 接口建议（第一期）

接口方法说明
/biz/loan/pageGET贷款申请分页查询
/biz/loan/detailGET贷款申请详情
/biz/loan/nodeTimelineGET节点进度时间线

8. 本期不做

  • 审核动作（通过/驳回）
  • 自动风控评分
  • 放款指令联动
  • 贷后风险预警联动

9. 验收标准（最小）

  1. 可按条件检索贷款申请记录。
  2. 可查看单条申请完整信息与流程节点状态。
  3. 权限隔离生效：不同农场用户仅见授权数据。
