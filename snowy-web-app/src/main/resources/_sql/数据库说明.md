# 数据库说明
1.snowy_mysql.sql：mysql版本的主sql文件，如果你是mysql，运行此sql并启动项目即可
2.snowy_postgresql.sql：postgresql版本的主sql文件，如果你是postgresql，运行此sql并启动项目即可
3.exm_flw_leave.sql：请假申请示例的sql，用于测试工作流的表，可运行可不运行
4.exm_flw_car.sql：用车申请示例的sql，用于测试工作流的表，可运行可不运行

5.snowy_ten_mysql.sql：mysql版本的子租户sql文件，如果你是mysql，创建DB隔离类型的租户数据源时，运行此sql即可
6.snowy_ten_oracle.sql：oracle版本的子租户sql文件，如果你是oracle，创建DB隔离类型的租户数据源时，运行此sql即可

# 更新说明
update文件夹为更新文件，如果您是在更新平台版本，直接使用update下对应的数据库文件即可（注意，如果落后版本太多，需要将各个版本的sql 逐一运行）
